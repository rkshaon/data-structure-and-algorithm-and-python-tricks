# OOP in Python
Object oriented programming (OOP) in Python is a programming paradigm that uses objects and classes to model real-world entities and their relations. Some main features of OOP in Python are:
- Classes
- Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Modularity
